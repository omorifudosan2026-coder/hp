## 2. 会社概要（登記・業法に沿った表記）

`src/company.njk` の会社概要表を、**登記簿・宅建業法表示**と突き合わせて更新する。

- **会社名**（商号・法人種別の表記）
- **所在地**（本店所在地）
- **代表者名**（例: `代表取締役 大森 太郎` はダミー想定）
- **設立年月日・資本金**（実データと一致させる）
- **事業内容**（実際に行う業務に合わせる）
- **古物商・建設業許可** など、該当する場合は行を追加
- **宅地建物取引業者免許**（例: `東京都知事（1）第000000号` はダミー）
- **所属・加盟団体**（実在する場合のみ。ない場合は削除または「なし」等の扱いを法務と確認）
- **営業時間・定休日**（店舗・営業形態に合わせる）

## 5. お問い合わせフォーム

`src/contact.njk`

- フォームに **`action` や送信先 API がない**ため、送信してもサーバーに届かない状態の可能性が高い。本番では次のいずれか等で実装する:
  - Firebase（Cloud Functions + メール / Firestore）
  - 外部フォームサービス
  - メールサーバー連携
- 受付メールアドレス・自動返信文・個人情報の取り扱い（プライバシーポリシーとの整合）

---

## 6. SNS・外部リンク

`src/_includes/layouts/base.njk` フッター

- Facebook / Instagram アイコンは `title` で「準備中」とあり、リンク先が **`/contact.html`** のまま。公式アカウント URL が決まったら差し替え。運用しない場合はアイコン削除も検討。

## 7. 画像・ブランドアセット

- **`/images/logo.png`**: 正式ロゴに差し替え（サイズ・余白の調整）

- **`src/concept.njk`**: サービス紹介のビジュアルがグラデーション枠のみの箇所は、実写真・図に差し替えを検討
